# 微信网页抓包调试指南

## 所需工具
- 一部手机
- 一台电脑
- 一条数据线

## 链接步骤

1.用数据线连接手机与电脑

2.手机打开 USB 调试

3.微信发送调试链接。
- 随便打开一个聊天窗口，输入并发送：
```
http://debugxweb.qq.com/?inspector=true
```
- 点击打开该链接，弹出“执行成功”即可

4.微信打开需要抓包调试的网页

5.电脑端打开浏览器
- Chrome 浏览器输入：
```
chrome://inspect/#devices
```
- Edge 浏览器输入：
```
edge://inspect/#devices
```

如果以上操作无误，您应当看到以下图片中的界面：下方为您的设备和您设备上打开的网页
到此，你就可以像在电脑浏览器的开发者工具中一样调试该网页
